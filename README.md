Gym Deck is a gamified workout planner built with Kotlin + Jetpack Compose.

## Features
- Exercise cards & decks
- Workout timer
- Ratings & stars
- Saved decks & history
- Firebase Auth & Firestore
- Room DB (offline support)
- Weekly activity strip

## Setup Instructions
1. Open project in Android Studio
2. Add `google-services.json` in `app/`
3. Run the app
